Floorhosting (Web Hosting Platform)

Live Site GitHub Repo

Floor Hosting is a professional hosting management platform and control panel that allows users to manage their hosting services and subscriptions. This is a frontend-focused project built with HTML5, CSS3, and Vanilla JavaScript, featuring a responsive design that works seamlessly across all devices. The platform demonstrates clean UI/UX practices with localStorage for demo functionality.

Key Features

  • Authentication: User registration, secure login/logout, password management, and profile customization.
  • Service Management: View active services, manage settings, renew subscriptions, and track service status.
  • Billing & Payments: View billing history, multiple payment methods (Credit Card, PayPal, Bank Transfer), add/remove payment methods, and download invoices.
  • User Dashboard: Centralized dashboard displaying services, quick stats, and account information.
  • Responsive Design: Mobile, tablet, and desktop optimized interface with smooth navigation.
  • Form Validation: Real-time form validation with professional error handling and user feedback.

Technologies Used

HTML5 CSS3 Vanilla JavaScript Tailwind CSS localStorage Responsive Design

Development Highlights

  • Built a completely responsive control panel that works perfectly on mobile, tablet, and desktop devices.
  • Implemented client-side form validation and input handling with real-time user feedback.
  • Created an organized folder structure with reusable components and shared assets for maintainability.
  • Utilized localStorage API for demo data persistence and user session management.
  • Integrated Tailwind CSS for modern styling and Ionicons for consistent iconography.
  • Set up Apache .htaccess configuration for URL rewriting and security best practices.
  • Designed an intuitive multi-page navigation system with smooth transitions and professional UI/UX.
